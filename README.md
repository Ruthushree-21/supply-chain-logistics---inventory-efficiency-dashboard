🚚 Supply Chain Logistics & Inventory Efficiency Dashboard
📊 Overview

This project is a Tableau-based Supply Chain Analytics Dashboard designed to monitor and optimize logistics performance, inventory efficiency, warehouse operations, and delivery KPIs.

It provides actionable insights to improve operational efficiency and reduce supply chain costs through interactive visual analytics.

🎯 Key Features
KPI Dashboard: Fill Rate, Inventory Turnover, Lead Time, On-Time Delivery
Transportation Cost trend analysis
Warehouse performance comparison across locations
Stockout risk heatmaps
Product-level sales & demand treemaps
Interactive filters for dynamic data exploration
🛠️ Tools & Technologies
Tableau
Microsoft Excel
Data Cleaning & Preparation
Calculated Fields
Data Visualization & Dashboard Design
📌 Business Impact
Identify stockout risks early
Improve logistics efficiency and delivery performance
Optimize warehouse utilization
Support data-driven supply chain decision-making
